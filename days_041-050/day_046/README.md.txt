# Day 46 — Web Scraping Project

This project is part of **Day 46** of the *100 Days of Code – Python* bootcamp.

The focus of this day is **web scraping** — extracting structured information
from a website using Python.

---

## What the Project Does

- Sends HTTP requests to a target website
- Parses HTML content
- Extracts specific data from the page
- Stores the extracted data in a file for later use

---

## Tools & Libraries Used

- Python
- `requests`
- `BeautifulSoup` (from `bs4`)

---

## Key Concepts Practiced

- HTTP requests
- HTML structure and parsing
- CSS selectors / tag-based extraction
- Working with external data sources
- File handling in Python

---

## Notes

- This project is intended for learning purposes
- The scraped data is used locally and not redistributed
- Code is kept simple and readable to focus on core concepts
